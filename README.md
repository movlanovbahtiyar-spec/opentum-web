# Opentum Promo - Nasıl Kullanılır?

Tanıtım videonuzu oluşturmak için hazırladığım web tabanlı promo aracını şu adımlarla kullanabilirsiniz:

## 1. Dosyayı Açın
Şu adresteki dosyayı tarayıcınızda (Chrome önerilir) açın:
`/Users/bahtiyar/uygulama/rezervesistem/opentum-promo/index.html`

## 2. Ekran Görüntülerini Ekleyin
Sağladığınız 5 ekran görüntüsü için `assets` klasörüne şu isimlerle dosyaları kaydedin:
- **dashboard.png**: Dashboard (Status Cards)
- **analytics.png**: Dashboard (Charts/Analytics)
- **menu.png**: Sidebar Menu
- **shifts.png**: Shift Planning Screen
- **staff.png**: Staff Management Screen

### Nasıl eklenir?
Ekran görüntülerinizi `/Users/bahtiyar/uygulama/rezervesistem/opentum-promo/assets/` klasörüne yukarıdaki isimlerle kaydedin. 

Veya tarayıcı konsolundan şu komutu kullanabilirsiniz:
`window.setScreenshot('screen-dashboard', 'assets/dashboard.png')`

## 3. Kayıt Alın
1. Tarayıcıyı tam ekran yapın (F11 veya Chrome'da Full Screen).
2. **"Tanıtımı Oynat"** butonuna basın.
3. OBS, QuickTime veya benzeri bir programla ekranı kaydedin.
4. Animasyonlar bittiğinde kaydı durdurun.

Bu yöntemle profesyonel, 60 FPS bir uygulama tanıtım videosu elde etmiş olacaksınız.
